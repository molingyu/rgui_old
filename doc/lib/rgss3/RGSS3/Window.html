    class RGSS3::Window

    游戏内窗口的类，内部由多个精灵构成。 超类：Object。

    Attributes

    active[RW]

    光标闪烁的状态。true 表示闪烁。默认为 true。

    arrows_visible[RW]

    滚动箭头可见的状态，true 表示可见。默认为 true。

    back_opacity[RW]

    窗口背景的不透明度（0～255）。超出此范围的数值会自动修正。默认为 192。(RGSS3)

    contents[RW]

    窗口内容位图（Bitmap）的引用。

    contents_opacity[RW]

    窗口内容的不透明度（0～255）。超出此范围的数值会自动修正。默认为 255。

    cursor_rect[RW]

    光标矩形（Rect）。

    height[RW]

    窗口的高度。

    opacity[RW]

    窗口的不透明度（0～255）。超出此范围的数值会自动修正。默认为 255。

    openness[RW]

    窗口的打开程度（0～255）。超出此范围的数值会自动修正。 将此数值从 0（完全关闭）至 255（完全打开）之间变换，可以产生窗口打开和关闭的动画效果。 openness 不满 255 时，窗口的内容不会显示。默认为 255。

    ox[RW]

    窗口内容原点的 X 坐标。修改此数值可以滚动窗口。 该属性同样影响光标。

    oy[RW]

    窗口内容原点的 Y 坐标。修改此数值可以滚动窗口。 该属性同样影响光标。

    padding[RW]

    窗口边框与内容之间的边距，默认为 12。(RGSS3)

    padding_bottom[RW]

    底端用的 padding。由于该属性会随着 padding 的改变而改变，因此必须在 padding 之后设置。

    pause[RW]

    暂停标记可见的状态。暂停标记是表示消息窗口等待输入的记号。true 表示可见。默认为 false。

    tone[RW]

    窗口背景的色调（Tone）。

    viewport[RW]

    窗口关联的显示端口（Viewport）的引用。

    visible[RW]

    窗口可见的状态。true 表示可见。默认为 true。

    width[RW]

    窗口的宽度。

    windowskin[RW]

    窗口皮肤位图（Bitmap）的引用。

    x[RW]

    窗口的 X 坐标。

    y[RW]

    窗口的 Y 坐标。

    z[RW]

    窗口的 Z 坐标。数值越大的窗口显示在越前方。 Z 坐标相同的，越晚生成的对象显示在越前方。 默认为 100。(RGSS3)
